DLP & FDM Hybrid 3D Printer

Electronics Innovation

Project Description

This innovative hybrid 3D printer combines Digital Light Processing (DLP) and Fused Deposition Modeling (FDM) technologies to leverage the strengths of both approaches. DLP provides high-resolution printing for detailed components, while FDM offers material versatility and structural strength.

Technical Specifications

  • DLP Resolution: 50μm XY resolution with 405nm UV light source
  • FDM Nozzle: 0.4mm hardened steel nozzle (up to 300°C)
  • Build Volume: 120×120×150mm (DLP), 200×200×200mm (FDM)
  • Control System: Custom firmware based on Marlin with mode switching
  • Materials: Photopolymer resins (DLP), PLA/ABS/PETG (FDM)

Key Features

Dual-Mode Printing

Seamlessly switch between DLP and FDM based on part requirements

High Precision

50μm resolution for detailed features when using DLP mode

Material Versatility

Wide range of compatible materials for different applications

Cost Effective

Combined system more affordable than separate high-end printers

Future Scope

This hybrid approach opens new possibilities in additive manufacturing:

  • Multi-Material Printing: Combining resin and filament in single prints
  • Education: Ideal for engineering colleges teaching additive manufacturing
  • Medical Applications: High-detail surgical guides with structural supports
  • Commercial Potential: Startup opportunity for customized 3D printing solutions
  • Research: Platform for developing new composite printing techniques

FPGA PRESENT Cipher Implementation

Electronics Security

Project Description

This project implements the PRESENT lightweight encryption algorithm on FPGA hardware, creating an ultra-efficient cryptographic solution for IoT devices. The design achieves record-low power consumption while maintaining high throughput, making it ideal for edge devices with limited resources.

Technical Specifications

  • FPGA Platform: Xilinx Artix-7 (XC7A35T)
  • Encryption Standard: PRESENT-80 (Lightweight)
  • Throughput: 200Mbps at 100MHz clock
  • Power Consumption: 12mW during operation
  • Area Utilization: 850 LUTs, 420 FFs

Key Features

Strong Security

80-bit key with 31-round SPN structure

Low Power

Optimized for battery-powered devices

High Speed

200Mbps throughput suitable for real-time

Small Footprint

Minimal FPGA resources used

Future Scope

This cryptographic engine has significant potential:

  • IoT Security: Protection for smart devices and sensors
  • ASIC Implementation: Further optimization for mass production
  • Post-Quantum: Hybrid cryptographic solutions
  • Government: Secure communications for defense
  • Blockchain: Lightweight node security
